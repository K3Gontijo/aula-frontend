Tags de Vídeo

Para utilizarmos vídeos no nosso site, funciona básicamente da mesma forma que os áudios. Para acrescentarmos um novo, usamos a tag "video" e dentro dela colocamos uma subtag "source" com atributos "src"(podemos colocar o caminho direito do nosso pc e/ou algum link) e "type"(para referenciar qual formato de vídeo está sendo usado)

Automaticamente ele vai preencher toda a tela, para mudarmos isso podemos adicionar o atributo "width" na tag "video" para ajustarmos o tamanho dele.
Também precisamos adicionar os controles nele, só adicionarmos o atributo "controls"

Podemos remover a opção de download, fullscreen e picture-in-picture com os atributos "controlsList='nodownload'" (para remover a tela cheia, só adicionarmos "nofullscreen") e para removermos o picture-in-picture só colocarmos o atributo "disablepictureinpicture"

Podemos adicionar um atributo uma thumbnail, para que tenha uma imagem de capa do vídeo, esse atributo é o "poster". Com ele podemos adicionar um endereço de uma imagem para que seja carregada

Outra coisa que podemos adicionarm também, é fazer com que o vídeo seja carregado assim que a pessoa acesse a página. Só basta adicionarmos o atributo "autoplay". Em alguns navegadores essa função é bloqueada
